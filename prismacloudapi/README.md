# Python SDK for the Prisma Cloud APIs

This is a Python SDK for the Prisma Cloud APIs (CSPM, CWPP, and PCCS) in the form of a Python package.


## Table of Contents

* [Setup](#Usage)
* [Support](#Support)
* [References](#References)


## Setup

Install the SDK via:

```
pip3 install prismacloud-api
```


## Support
